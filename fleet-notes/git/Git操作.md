# 撤销已 `add` 的文件

```shell
git status

On branch main

No commits yet

Changes to be committed:
  (use "git rm --cached <file>..." to unstage)
        new file:   .gitignore
        new file:   pom.xml

```

使用 `git status` 命令后按照提示操作。
