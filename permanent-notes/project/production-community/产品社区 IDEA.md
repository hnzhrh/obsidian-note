# 参与方

供应商，supplier
需求方，seeker

# 痛点

找不到产品（软件、解决方案）

# 初步需求

供应商录入产品，可以设置分类、设置标签、设置产品名称、设置联系方式（座机、手机、微信、QQ、邮箱）、设置简要描述

需求方检索产品，可以根据类型（软件、解决方案）、类目、标签进行检索
